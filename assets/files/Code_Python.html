List

Python List Functions

Basic Operations

  • append(item): Adds an element to the end of the list.
  • clear(): Removes all elements from the list.
  • copy(): Returns a shallow copy of the list.
  • count(element): Returns the number of occurrences of an element in the list.
  • extend(iterable): Appends the elements of another iterable to the end of the list.
  • index(element, start=0, end=None): Returns the index of the first occurrence of an element in the list.
  • insert(index, element): Inserts an element at a specified position.
  • pop(index=None): Removes and returns the element at a specified position. If no index is specified, removes the last element.
  • remove(element): Removes the first occurrence of an element in the list.
  • reverse(): Reverses the order of the elements in the list.
  • sort(key=None, reverse=False): Sorts the elements in the list.

Slicing

  • list[start:stop:step]: Returns a new list containing a slice of the original list.

Dictionary (Map)

Python Dictionary Functions

Basic Operations

  • clear(): Removes all items from the dictionary.
  • copy(): Returns a shallow copy of the dictionary.
  • get(): Returns the value for a key, or a default value if the key is not found.
  • items(): Returns a list of tuples, each containing a key-value pair.
  • keys(): Returns a list of all keys in the dictionary.
  • values(): Returns a list of all values in the dictionary.
  • pop(): Removes a specified item and returns its value.
  • popitem(): Removes and returns an arbitrary item as a tuple.
  • setdefault(): Inserts a key with a specified default value if it doesn’t exist.
  • update(): Updates the dictionary with the contents of another dictionary or iterable.

Example Usage:

my_dict = {'apple': 10, 'banana': 20}

# Access elements
print(my_dict['apple'])

# Modify elements
my_dict['orange'] = 30

# Add elements
my_dict['grape'] = 40

# Remove elements
my_dict.pop('banana')

# Iterate over items
for key, value in my_dict.items():
    print(key, value)

Deque

Python Deque (Double-Ended Queue) Functions

Basic Operations

  • append(item): Adds an element to the right end (rightmost element).
  • appendleft(item): Adds an element to the left end (leftmost element).
  • clear(): Removes all elements from the deque.
  • count(element): Returns the number of occurrences of an element in the deque.
  • extend(iterable): Adds elements from another iterable to the right end.
  • extendleft(iterable): Adds elements from another iterable to the left end.
  • pop(): Removes and returns the rightmost element.
  • popleft(): Removes and returns the leftmost element.
  • rotate(n=1): Rotates the elements in the deque by n positions.

Other Methods

  • copy(): Returns a shallow copy of the deque.
  • index(element, start=0, stop=None): Returns the index of the first occurrence of an element.
  • insert(index, element): Inserts an element at a specified position.
  • remove(value): Removes the first occurrence of a value.
  • reverse(): Reverses the order of the elements.

Example Usage:

from collections import deque

queue = deque()

# Add elements to the right end
queue.append(1)
queue.append(2)
queue.append(3)

# Add elements to the left end
queue.appendleft(0)

# Remove and print elements
while queue:
    print(queue.popleft())

Heap

Python Heap Functions

Heap: A heap is a specialized tree-based data structure that satisfies the heap property: for every node, the value of the node is greater than or equal to (in a max heap) or less than or equal to (in a min heap) the values of its children.

Python’s standard library provides the heapq module for working with heaps.

Basic Operations

  • heapq.heapify(iterable): Converts an iterable into a heap in-place.
  • heapq.heappush(heap, item): Pushes an item onto the heap.
  • heapq.heappop(heap): Pops the smallest item from the heap.
  • heapq.nlargest(n, iterable, key=None): Returns the n largest elements from the iterable.
  • heapq.nsmallest(n, iterable, key=None): Returns the n smallest elements from the iterable.

Additional Functions

  • heapq.merge(*iterables, key=None): Merges multiple sorted iterables into a single sorted iterable.
  • heapq.heapreplace(heap, item): Pops the smallest item from the heap and pushes the new item onto the heap.

Example:

import heapq

# Create a max heap
max_heap = [3, 2, 1, 5, 4]
heapq.heapify(max_heap)

# Push an item onto the heap
heapq.heappush(max_heap, 6)

# Pop the smallest item from the heap
smallest = heapq.heappop(max_heap)

print(max_heap)
print(smallest)
