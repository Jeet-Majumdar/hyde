Array

Creating Arrays

  • new Type[size]: Creates a new array of the specified type and size.
  • Type[] arrayName = {element1, element2, ...}: Creates an array with initial values.

Accessing Elements

  • arrayName[index]: Accesses the element at the specified index.

Modifying Elements

  • arrayName[index] = newValue: Modifies the element at the specified index.

Array Length

  • arrayName.length: Returns the length of the array.

Copying Arrays

  • System.arraycopy(src, srcPos, dest, destPos, length): Copies elements from one array to another.
  • Arrays.copyOf(array, newLength): Returns a copy of the array with a new length.
  • Arrays.copyOfRange(array, fromIndex, toIndex): Returns a copy of a subarray.

Sorting Arrays

  • Arrays.sort(array): Sorts the elements in the array in ascending order.
  • Arrays.sort(array, Comparator<T> comparator): Sorts the elements in the array using a custom comparator.

Searching Arrays

  • Arrays.binarySearch(array, key): Searches for a key in a sorted array using binary search.
  • Arrays.binarySearch(array, fromIndex, toIndex, key): Searches for a key in a sorted subarray using binary search.

Filling Arrays

  • Arrays.fill(array, value): Fills the array with a specified value.
  • Arrays.fill(array, fromIndex, toIndex, value): Fills a subarray with a specified value.

Comparing Arrays

  • Arrays.equals(array1, array2): Compares two arrays for equality.

Example Usage:

import java.util.Arrays;

public class ArrayExample {
    public static void main(String[] args) {
        int[] numbers = new int[5];

        // Assign values
        numbers[0] = 10;
        numbers[1] = 20;
        numbers[2] = 30;
        numbers[3] = 40;
        numbers[4] = 50;

        // Access an element
        System.out.println(numbers[2]);

        // Modify an element
        numbers[1] = 25;

        // Get the array length
        System.out.println("Array length: " + numbers.length);

        // Sort the array
        Arrays.sort(numbers);

        // Search for an element
        int index = Arrays.binarySearch(numbers, 30);
        System.out.println("Index of 30: " + index);

        // Fill the array with a value
        Arrays.fill(numbers, 0);

        // Print the array
        System.out.println(Arrays.toString(numbers));
    }
}

Map, TreeMap, & HashMap

Basic Operations

  • clear(): Removes all elements from the map.
  • containsKey(key): Checks if the map contains the specified key.
  • containsValue(value): Checks if the map contains the specified value.
  • entrySet(): Returns a set of all entries in the map.
  • get(key): Returns the value associated with the specified key, or null if the key is not found.
  • isEmpty(): Checks if the map is empty.
  • keySet(): Returns a set of all keys in the map.
  • put(key, value): Associates the specified value with the specified key.
  • putAll(map): Copies all key-value pairs from another map into this map.
  • remove(key): Removes the mapping for the specified key if present.
  • size(): Returns the number of key-value pairs in the map.
  • values(): Returns a collection of all values in the map.

Specific Implementations

HashMap:

  • Unordered hash table implementation.
  • Fast for most operations.
  • Allows null keys and values.

TreeMap:

  • Red-black tree implementation.
  • Sorts keys in ascending order.
  • Does not allow null keys.

LinkedHashMap:

  • Extends HashMap and maintains insertion order or access order.
  • Useful for caching or maintaining order.

Example Usage:

import java.util.HashMap;
import java.util.Map;

public class MapExample {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();

        // Add elements
        map.put("apple", 10);
        map.put("banana", 20);
        map.put("orange", 30);

        // Get a value
        int appleCount = map.get("apple");
        System.out.println("Apple count: " + appleCount);

        // Check if a key exists
        boolean containsBanana = map.containsKey("banana");
        System.out.println("Contains banana: " + containsBanana);

        // Iterate over key-value pairs
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            String key = entry.getKey();
            int value = entry.getValue();
            System.out.println(key + ": " + value);
        }
    }
}

Set

Java Set Functions

Basic Operations

  • add(element): Adds an element to the set.
  • clear(): Removes all elements from the set.
  • contains(element): Checks if the set contains the specified element.
  • isEmpty(): Checks if the set is empty.
  • remove(element): Removes the specified element from the set.
  • size(): Returns the number of elements in the set.

Iterators

  • iterator(): Returns an iterator over the elements in the set.
  • forEach(action): Performs an action for each element in the set.

Additional Operations

  • addAll(collection): Adds all elements from the specified collection to the set.
  • containsAll(collection): Checks if the set contains all elements from the specified collection.
  • removeAll(collection): Removes all elements from the set that are contained in the specified collection.
  • retainAll(collection): Retains only the elements in the set that are also in the specified collection.

Example Usage:

import java.util.HashSet;
import java.util.Set;

public class SetExample {
    public static void main(String[] args) {
        Set<String> mySet = new HashSet<>();

        // Add elements
        mySet.add("apple");
        mySet.add("banana");
        mySet.add("orange");

        // Check if an element exists
        if (mySet.contains("apple")) {
            System.out.println("Set contains apple");
        }

        // Remove an element
        mySet.remove("banana");

        // Iterate over elements
        for (String element : mySet) {
            System.out.println(element);
        }
    }
}

Stack

Java Stack Methods

Basic Operations

  • empty(): Checks if the stack is empty.
  • peek(): Returns the object at the top of the stack without removing it.
  • pop(): Removes the object at the top of the stack and returns it.
  • push(Object item): Pushes an item onto the top of the stack.
  • search(Object o): Returns the 1-based position of the object in the stack. Returns -1 if the object is not found.

Example Usage:

import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        Stack<String> stack = new Stack<>();

        stack.push("apple");
        stack.push("banana");
        stack.push("orange");

        System.out.println("Top element: " + stack.peek());

        System.out.println("Popped element: " + stack.pop());

        System.out.println("Is the stack empty? " + stack.empty());
    }
}

Queue

Java Queue Functions

Basic Operations

  • add(E e): Inserts the specified element into this queue if it is possible to do so immediately.
  • offer(E e): Inserts the specified element into this queue if it is possible to do so immediately. Returns true if the element was added to the queue, false otherwise.
  • remove(): Retrieves and removes the head of this queue.
  • poll(): Retrieves and removes the head of this queue, or null if this queue is empty.
  • element(): Retrieves, but does not remove, the head of this queue.
  • peek(): Retrieves, but does not remove, the head of this queue, or null if this queue is empty.

Additional Operations

  • size(): Returns the number of elements in this queue.
  • isEmpty(): Returns true if this queue contains no elements.
  • contains(Object o): Returns true if this queue contains the specified element.
  • iterator(): Returns an iterator over the elements in this queue.

Example Usage:

import java.util.LinkedList;
import java.util.Queue;

public class QueueExample {
    public static void main(String[] args) {
        Queue<Integer> queue = new LinkedList<>();

        // Add elements to the queue
        queue.add(10);
        queue.add(20);
        queue.add(30);

        // Retrieve and remove the head
        int removedElement = queue.remove();
        System.out.println("Removed element: " + removedElement);

        // Peek at the head without removing
        int headElement = queue.peek();
        System.out.println("Head element: " + headElement);

        // Check if the queue is empty
        boolean isEmpty = queue.isEmpty();
        System.out.println("Queue is empty: " + isEmpty);
    }
}

Deque

Java Deque Functions

Basic Operations

  • addFirst(E e): Inserts an element at the front of the deque.
  • addLast(E e): Inserts an element at the end of the deque.
  • getFirst(): Retrieves the first element of the deque without removing it.
  • getLast(): Retrieves the last element of the deque without removing it.
  • removeFirst(): Removes and returns the first element of the deque.
  • removeLast(): Removes and returns the last element of the deque.
  • peekFirst(): Retrieves the first element of the deque without removing it, or null if the deque is empty.
  • peekLast(): Retrieves the last element of the deque without removing it, or null if the deque is empty.
  • size(): Returns the number of elements in the deque.
  • isEmpty(): Checks if the deque is empty.

Additional Operations

  • offerFirst(E e): Inserts an element at the front of the deque if possible, returning true if successful or false if the deque is full.
  • offerLast(E e): Inserts an element at the end of the deque if possible, returning true if successful or false if the deque is full.
  • pollFirst(): Removes and returns the first element of the deque, or null if the deque is empty.
  • pollLast(): Removes and returns the last element of the deque, or null if the deque is empty.

Example Usage:

import java.util.Deque;
import java.util.LinkedList;

public class DequeExample {
    public static void main(String[] args) {
        Deque<String> deque = new LinkedList<>();

        // Add elements
        deque.addFirst("A");
        deque.addLast("B");
        deque.addLast("C");

        // Access elements
        System.out.println(deque.getFirst()); // Output: A
        System.out.println(deque.getLast()); // Output: C

        // Remove elements
        deque.removeFirst();
        deque.removeLast();

        // Check if empty
        System.out.println(deque.isEmpty()); // Output: true
    }
}

Heap

Java Heap Data Structure

Heap is a complete binary tree with the property that the value of any node is greater than (or less than) the values of its children. There are two types of heaps:

  • Max heap: The value of any node is greater than or equal to the values of its children.
  • Min heap: The value of any node is less than or equal to the values of its children.

Note: Java doesn’t have a built-in heap class, but you can implement it using PriorityQueue class.

PriorityQueue Class Methods

Basic Operations

  • PriorityQueue(): Creates an empty priority queue.
  • PriorityQueue(Collection<? extends E> c): Creates a priority queue containing the elements of the specified collection.
  • PriorityQueue(int initialCapacity): Creates a priority queue with the specified initial capacity.
  • PriorityQueue(int initialCapacity, Comparator<? super E> comparator): Creates a priority queue with the specified initial capacity and comparator.
  • add(E e): Inserts the specified element into this priority queue.
  • offer(E e): Inserts the specified element into this priority queue if it is possible to do so immediately without violating the capacity restrictions.
  • poll(): Retrieves and removes the head of this queue, or returns null if this queue is empty.
  • remove(): Removes the head of this queue.
  • peek(): Retrieves, but does not remove, the head of this queue, or returns null if this queue is empty.
  • size(): Returns the number of elements in this queue.

Additional Operations

  • comparator(): Returns the comparator used to order the elements in this queue, or null if the natural ordering is used.
  • contains(Object o): Returns true if this queue contains the specified element.
  • toArray(): Returns an array containing all of the elements in this queue.
  • toArray(T[] a): Returns an array containing all of the elements in this queue, using the given array as much as possible.

Example Usage:

import java.util.PriorityQueue;

public class HeapExample {
    public static void main(String[] args) {
        PriorityQueue<Integer> maxHeap = new PriorityQueue<>(10, (a, b) -> b - a); // Create a max heap

        maxHeap.offer(5);
        maxHeap.offer(2);
        maxHeap.offer(8);
        maxHeap.offer(1);
        maxHeap.offer(9);
        maxHeap.offer(3);

        System.out.println("Max Heap: " + maxHeap);

        // Remove the maximum element
        System.out.println("Removed: " + maxHeap.poll());

        // Check if the heap is empty
        System.out.println("Is empty: " + maxHeap.isEmpty());
    }
}
